env myenv\Scripts\activate 

1. Instalar Django via Anaconda 
2. Abrir Anaconda Prompt 
3. Navegar até a pasta que tem o arquivo manage.py 
4. Rodar o banco: python manage.py makemigrations blog python manage.py migrate blog 
5. Rodar o servidor: python manage.py runserver 

OBS: Eu não deixo o Banco Final no git para segurança (substituir o único arquivo)
